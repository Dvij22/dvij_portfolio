# Dvij Pandya Portfolio

This is my personal portfolio website built with **React** and **Vite**, showcasing my projects, skills, and contact information. It uses modern front-end tools and libraries like **Framer Motion** for animations and **EmailJS** for the contact form.

## Features

- Fully responsive design
- Smooth animations with [Framer Motion]
- Contact form powered by [EmailJS]
- Interactive project showcase
- Social media links (GitHub, LinkedIn)

## Tech Stack

- **React** – Front-end library for building UI
- **Vite** – Build tool with fast development server
- **TailwindCSS** – Utility-first CSS framework
- **Framer Motion** – Animations
- **EmailJS** – Sending messages from the contact form
- **GitHub Pages** – Deployment options
